Minecraft: Delta Edition

A mod for Minecraft Alpha v1.1.2_01

Download

Features

- Download Prism or MultiMC
- Create instance of Alpha v1.1.2_01
- Edit instance
- Go to version section
- Click "Add to Minecraft.jar"
- Select the zip file you downloaded
- Have fun!

- Adds a "pretty visuals" mode (toggleable) that changes foliage colors, sky and fog colors, and water color
- Adds a brightness slider that clamps how dark shadows and caves can be in rendering
- Adds an FOV slider
- Makes the F3 toggle rather than the user having to hold down F3
- Adds player coordinates to F3 menu
- Makes the alpha watermark toggleable
- Binds F1 to rotate between hidden UI, hidden hand and hidden UI, and standard visibility
- Backports screenshot feature from later alpha versions
- Changes main menu background to cobblestone
- Sets default inventory keybind to E
